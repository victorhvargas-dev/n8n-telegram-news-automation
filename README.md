## Descripción del Proyecto

Este proyecto implementa un sistema de automatización utilizando n8n que permite gestionar noticias a través de un bot de Telegram, integrando almacenamiento en Google Sheets, flujos de aprobación y consumo de APIs externas.

El sistema permite a los usuarios:

* Registrar noticias mediante comandos en Telegram
* Enviar contenido para revisión
* Aprobar o rechazar noticias mediante un flujo automatizado
* Consultar noticias aprobadas en tiempo real
* Obtener información adicional como precios de gasolina y diésel desde APIs externas

El flujo está diseñado para simular un entorno real de operación, donde se requiere control de acceso, validación de datos y automatización de procesos manuales.

---

## Objetivo

Demostrar la implementación de automatización de procesos empresariales mediante integración de herramientas, reduciendo intervención manual y mejorando la eficiencia operativa.

---

## Valor del Proyecto

Este workflow refleja un caso real de negocio donde se integran múltiples servicios para crear una solución automatizada completa:

* Automatización de procesos
* Integración de sistemas
* Manejo de datos en tiempo real
* Interacción con usuarios mediante bots
* Consumo y procesamiento de APIs
